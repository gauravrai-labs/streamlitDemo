# Streamlit Demo

A comprehensive Streamlit learning and demo project showcasing various features and functionalities of the Streamlit framework.

## Project Overview

This project is designed to demonstrate the basics and practical implementations of Streamlit, a popular Python library for building interactive web applications. It includes various demo pages covering different use cases and features.

## Features

- **Main Application**: Basic Streamlit interface with welcome screen
- **Basic Syntax Demo**: Examples of Streamlit's fundamental components and syntax
- **Currency Converter page**: Interactive currency conversion tool built with Streamlit
- **Pandas Demo**: Data manipulation and visualization using pandas with Streamlit

## Requirements

- Python 3.7+
- streamlit
- pandas

## Installation

1. Clone or download this project
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the main application:
```bash
streamlit run main.py
```

The application will open in your default web browser at `http://localhost:8501`

### Navigation

- Use the sidebar to navigate between different demo pages
- Each page demonstrates different Streamlit features and use cases

## Getting Started

This project is ideal for:
- Learning Streamlit fundamentals
- Exploring interactive web app development with Python
- Understanding data visualization with Streamlit and pandas
- Building simple tools like currency converters
